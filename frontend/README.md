# ApartemenKu Frontend

## installation

```sh
npm install
```

Craete and set env backend

```env
VITE_BACKEND_URL="http://127.0.0.1:8000/api"
```

Running Frontend

```sh
npm run dev
```

Build Frontend

```sh
npm run build
```
